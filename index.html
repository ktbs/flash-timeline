flash-timeline

A generic timeline for visualising modelled traces, coded in flash.

This software is a flash application whose purpose is to provide a generic timeline visualisation for modelled activity traces.

Dependencies

It depends on the com.ithaca.traces package, which can be obtained from the http://github.com/oaubert/astraces1 repository (Traces.swc component). The Traces.swc file must be placed (or linked) in the lib/ directory for compilation of the Timeline.swc component.

Install

Example applications using the timeline component are present in the examples directory.
